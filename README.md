# 🌤️ 彩云笔记 (CaiYun Notes)

> 一款为个人知识管理设计的全栈笔记应用，支持富文本编辑、多页签、思维导图等功能。

在线体验：[http://49.235.160.8:3011/](http://49.235.160.8:3011/)

---

## ✨ 功能特性

### 📝 富文本编辑
基于 TipTap (ProseMirror) 的所见即所得编辑器，覆盖日常写作所需的全部排版能力：
- 标题（H1-H3）、段落、引用块、代码块
- 有序列表、无序列表、待办事项（圆形复选框）
- 粗体、斜体、删除线、下划线、高亮标记
- 表格（增删行列、合并单元格、列宽拖拽）
- 图片（拖拽上传、调整大小、居左/居中/居右对齐）
- 思维导图（MindElixir 集成，节点增删、缩放平移、导出图片/Markdown）
- 选中内容一键复制为高清图片

### 📑 多页签
每个笔记支持创建多个页签（Tab），在不同页签中独立编写内容。适合同一主题下的多维度记录场景。

### 📁 笔记本管理
- 树形笔记本结构，无限层级嵌套
- 拖拽排序，快速调整笔记和分区位置
- 侧边栏折叠/展开，状态自动保存
- 笔记锁定，防止他人误编辑

### 🔄 导入导出
- JSON 格式导入导出，保留完整树形结构和父子关系
- ID 冲突自动处理（导入到已有笔记本时智能重映射）
- 支持导入到原位置或指定目标

### 🔐 账户与安全
- 邮箱注册（验证码验证）、登录、密码重置
- JWT + bcrypt 认证，密钥环境变量管理
- 管理后台（用户管理、笔记管理、数据统计）

### 💻 桌面客户端 (Tauri)
- 支持 Windows / macOS (Intel + Apple Silicon) / Linux
- 应用内自动检测并安装更新
- 本地文件系统备份
- 原生剪贴板图片写入

## 🛠️ 技术栈

| 层级 | 技术选型 |
|------|----------|
| 前端框架 | React 18 + TypeScript + Vite |
| 富文本引擎 | TipTap (ProseMirror) |
| 思维导图 | MindElixir |
| 桌面端 | Tauri v2 (Rust) |
| 后端 | Node.js (原生 http 模块) |
| 数据库 | PostgreSQL |
| 认证 | JWT (HMAC-SHA256) + bcrypt |
| 邮件服务 | nodemailer |
| 反向代理 | Nginx |
| 进程管理 | PM2 |
| CI/CD | GitHub Actions (多平台自动构建) |
